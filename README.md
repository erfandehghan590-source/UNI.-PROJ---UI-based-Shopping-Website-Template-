# UI-based Shopping Website Template
A simple and user-friendly shopping website template with a Flask backend and SQLite database. This project was developed as a team effort for a university course. I was responsible for the HTML, CSS, and JavaScript parts.

## Features
- Responsive and attractive UI built with HTML/CSS/JS
- Flask API for backend communication
- SQLite database for data management

## Project Structure
├── API/
├── database/
├── static/
├── templates/

├── database.db
├── LICENSE
└── README.md

## How to Run
1. Clone the repository
git clone https://github.com/erfandehghan590-source/UNI-PROJ-UI-based-Shopping-Website-Template.git
cd UNI-PROJ-UI-based-Shopping-Website-Template
2. Create a virtual environment (recommended)
python -m venv venv
source venv/bin/activate   # On Linux/Mac
venv\Scripts\activate      # On Windows
3. Install dependencies
pip install flask
4. Run the project
python shop.py
5. Open in your browser
http://127.0.0.1:5000
